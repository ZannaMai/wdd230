Typography

Heading Font: Dancing Script

Dancing Script is a good font to use because it looks professional but is still different enough that it makes the website interesting and helps it have a somewhat different feel. Because it is harder to read though, it is only used for the headings.

Paragraph Font: Garamond

Garamond is a very simple serif font that is easy to read. It goes well with the headings and adds the needed contrast. In large paragraphs though, it is simple and clean.
